# UI Copy: MVP2 口语化文案

## 按钮 (Buttons)
- **开始修复**: “陈老师，带我搞定这门题”
- **解锁按钮**: “输入暗号，开启 7 天特训”
- **提交判定**: “我写好了，陈老师帮我看看”
- **去复检**: “成果验收：看看我进步了多少”
- **升级引导**: “搞定一类题不休息，我要搞定这一学期”

## 提示语 (Prompts)
- **解锁提示**: “这个是付费进阶内容哦，请输入你的 7 天邀请码。”
- **每日任务**: “Day {{n}}：今天我们要攻克的关卡是——{{task_name}}。准备好了吗？”
- **提交提示**: “别担心做得全不全，把目前写的拍下来给我就行。我来看看你卡在哪了。”
- **复检结论**: “漂亮！原本卡住的那一步，你现在已经能行云流水地写出来了。”
- **429 退避**: “陈老师正在思考中，稍微等我 2 秒钟，正在帮你排队...”
