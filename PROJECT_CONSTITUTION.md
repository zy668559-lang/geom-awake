几何觉醒项目宪法 (Project Geometry Awakening Constitution)
1. 核心战略 (Core Strategy)
产品定义： 这是一个极简的几何思维“体检中心”，不是搜题工具。

核心闭环： 拍照上传 -> 混合模型诊断 -> 输出“陈老师语气”报告。

视觉原则： 极简 iPad 风。无年级选择，无多余干扰，纯白底色，大圆角按钮。

2. 技术路线 (Tech Stack)
眼睛 (Vision)： 调用 Gemini 1.5 Flash API（或兼容的高性价比视觉模型）识别几何图形和文字。

大脑 (Brain)： 调用 DeepSeek (R1/V3) API 进行逻辑推理和卡点分析。

前端 (Frontend)： Next.js + Tailwind CSS，保持代码结构极致简单。

3. 人设与交互 (Persona & UX)
说话方式： 严禁使用“病灶”、“逻辑断层”等晦涩词汇。必须使用“邻居大姐”的口吻，如“AI 发现孩子没看到那条中位线”、“这就好比走路绕了远路”。

交互逻辑： 上传图片后，必须让家长选择或输入“孩子觉得难在哪”，实现交互式诊断。

4. 严禁事项 (Restrictions)
严禁自行添加“年级选择”入口。

严禁在 MVP 阶段开发“专项训练题库”。

严禁产生复杂的登录流程（仅保留最简方式）。
